About Me
This is a Java UDP Client Server program.
Server will make a port to listen for any wireless matching port.
Client will ask for user IPv4 address and port number to match with server port
After client wireless connect with the server, the client send the message to the server and will receive and respond back with capitalize letters

User Instruction

Open command prompt where server.java is located at and type
    javac server.java
    java server <Enter port number>

Open command prompt where client.java is located at and type
    javac client.java
    java client <Your IPv4 address> <Enter port number>
    Enter your response (Enter !exit to exit)
